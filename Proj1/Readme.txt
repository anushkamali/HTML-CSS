Proj1

This is a mini project build using HTML/CSS concepts. It is a website for travellers (WebsiteName), which allows them to look for information on destinations to travel, 
shop for accessories needed while travelling etc. It contains four pages:
1. Home (Provides information about website, news related to website)
2. Gallery (Picture gallery of travel destinations, click picture to know more about the particular destination)
3. Shop (Image gallery with pictures of travellers with accessories. Clicking accessory from the picture will navigate user to shopping website.)
4. About (Provides information about authors/founders and achievements of WebsiteName)
